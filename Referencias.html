Referências e citações1

Omar Cléo

1 Referências

Agora estamos próximo do final da nossa análise de dados e precisamos citar os artigos e livros que usamos para apoiar nosso estudo.

Citar suas referências em um documento rmarkdown é simples. Inicialmente você vai precisar criar um arquivo .bib. Um arquivo .bib armazena todas as referências do seu artigo e pode ser usado também no \(\LaTeX\), uma outra linguagem de marcação (e que não vem ao caso agora).

Para citar o software R no seu artigo, você pode usar o comando citation() no console do R.

citation()

To cite R in publications use:

  R Core Team (2021). R: A language and environment for statistical
  computing. R Foundation for Statistical Computing, Vienna, Austria.
  URL https://www.R-project.org/.

A BibTeX entry for LaTeX users is

  @Manual{,
    title = {R: A Language and Environment for Statistical Computing},
    author = {{R Core Team}},
    organization = {R Foundation for Statistical Computing},
    address = {Vienna, Austria},
    year = {2021},
    url = {https://www.R-project.org/},
  }

We have invested a lot of time and effort in creating R, please cite it
when using it for data analysis. See also 'citation("pkgname")' for
citing R packages.

Então você deve copiar o seguinte:

@Manual{,
    title = {R: A Language and Environment for Statistical Computing},
    author = {{R Core Team}},
    organization = {R Foundation for Statistical Computing},
    address = {Vienna, Austria},
    year = {2019},
    url = {https://www.R-project.org/},
  }

e salvar no seu arquivo .bib. Como outro exemplo, vamos citar o package nlme:

citation('nlme')

Pinheiro J, Bates D, DebRoy S, Sarkar D, R Core Team (2021). _nlme:
Linear and Nonlinear Mixed Effects Models_. R package version 3.1-152,
<URL: https://CRAN.R-project.org/package=nlme>.

A BibTeX entry for LaTeX users is

  @Manual{,
    title = {{nlme}: Linear and Nonlinear Mixed Effects Models},
    author = {Jose Pinheiro and Douglas Bates and Saikat DebRoy and Deepayan Sarkar and {R Core Team}},
    year = {2021},
    note = {R package version 3.1-152},
    url = {https://CRAN.R-project.org/package=nlme},
  }

E copiamos para o arquivo .bib o seguinte:

@Manual{,
    title = {{nlme}: Linear and Nonlinear Mixed Effects Models},
    author = {Jose Pinheiro and Douglas Bates and Saikat DebRoy and Deepayan Sarkar and {R Core Team}},
    year = {2020},
    note = {R package version 3.1-147},
    url = {https://CRAN.R-project.org/package=nlme},
  }

Uma forma interessante de referenciar um artigo é usar o Google Acadêmico (https://scholar.google.com.br/scholar?q=). Observa a Figura @ e veja onde está circulado em vermelho.

Clicando ali, aparecerá uma quadro chamado Citar (Figura 1.2).

@article{verbeke1996linear,
  title={A linear mixed-effects model with heterogeneity in the random-effects population},
  author={Verbeke, Geert and Lesaffre, Emmanuel},

Basta copiar e colar no seu documento .bib.

Depois de criado seu arquivo .bib com suas referências, você precisa chamá-lo no YAML do seu documento na opção bibliography. Suponha que você tenha dado o nome a esse arquivo de Referencias.bib, o YAML fica da seguinte forma:

---
title: 
author:
output: html_document
bibliography: Referencias.bib
---

1.1 Citação das referências

Para citar uma referência no texto você precisa do comando @ antes do nome do artigo ou livro. O nome da referência é você que escolhe. Observa que as referências do R e do package nlme começam da seguinte forma: @Manual{,. Entre a chave e a vírgula você deve acrescentar um nome. Por exemplo, para o R podemos colocar softwareR. Neste caso, a citação no arquivo .bib fica:

@Manual{softwareR,
    title = {R: A Language and Environment for Statistical Computing},
    author = {{R Core Team}},
    organization = {R Foundation for Statistical Computing},
    address = {Vienna, Austria},
    year = {2019},
    url = {https://www.R-project.org/},
  }

Para citá-lo no texto você deve digitar (@softwareR):

Esta análise estatística foi realizada por meio do software R (R Core Team (2019)).

E quando compilar seu documento, o seu texto ficarà assim:

Esta análise estatística foi realizada por meio do software R (Core Team (2019)).

E no final do seu documento aparecerá a citação:

R Core Team (2019). R: A language and environment for statistical computing. R
  Foundation for Statistical Computing, Vienna, Austria. URL
  https://www.R-project.org/.

Para mais detalhes sobre estilos de citação, ver os sites https://docs.citationstyles.org/en/1.0.1/primer.html e https://github.com/citation-style-language/styles

R Core Team. 2019. R: A Language and Environment for Statistical Computing. Vienna, Austria: R Foundation for Statistical Computing. https://www.R-project.org/.
